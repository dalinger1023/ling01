  • git第一次使用要初始化 git init
  • git第一次使用要设置用户信息--邮箱 git config --global user.email "you@example.com"
  • git第一次使用要设置用户信息--用户名 git config --global user.name "yourname"
  • 提交 git add 文件路径 git add . 当前文件夹全部提交 git commit -m "版本信息描述"
  • 一次性全部直接提交 git commit --all -m "版本信息描述" 修改备注的版本信息:git commit --amend
  • 查看版本信息 git log 查看简约版一行显示的版本信息 git log --oneline
  • 版本穿梭 git reset --hard Head~1回到上个版本 2表示上上个版本
  • 版本穿梭 git reset --hard [版本号] 回到指定版本
  • 版本回退后 查看所有的版本信息(版本切换记录) git reflog
  • 版本回退后 查看当前目中的文件的提交状态 git status
  • git status
    • untracked(红色文件名) 新建的文件,没有add过
    • modified(红色文件名) 修改过的文件,没有add过
    • modified(绿色文件名) 修改过的文件,已经add过staged,没有commit过
    • modified(绿色文件名) 修改过的文件,已经commit过committed
  • git的分支功能
    • git branch 查看所有分支
    • git branch [分支名] 新建一个分支
    • git checkout [分支名] 切换到指定分支
    • git checkout -b [分支名] 新建分支,并切换到该分支
    • git branch -d [分支名] 删除分支,不能在当前分支下删除当前分支
    • git merge [分支名] 合并分支 把[指定]分支合并到[当前]分支,不能在被合并的分支下进行这一步操作
